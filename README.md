​《NeoGenesis》是一款基于纯前端技术 (HTML/CSS/JS) 开发的赛博朋克风生存繁育模拟游戏。玩家将扮演一名认知滤网开启的 AI 管理员，通过基因融合、地表探索和资源管理，在末日废土中延续种群。
​✨ 核心特性 (Features)
​🧬 真实的遗传学算法： 引擎层 (engine.js) 实现了基于多基因座的显隐性遗传逻辑。包含近亲衰退判定、突变概率浮动以及 10+ 种复杂的基因缺陷表现。
​🌍 异步探险事件机： 动态计算辐射累积与生命值损耗，高级突变体（S/SS级）在探险时可触发专属技能（如立场免伤、资源双倍）。
​💻 Cyberpunk 视觉交互： 采用无框架原生 CSS 编写，包含动态星云背景、粒子动画、毛玻璃拟态面板 (Glassmorphism) 以及深度定制的霓虹发光系统。
​⚖️ 资源与状态机管理： 严格的容量限制与经济系统。阵亡个体会被保留为“墓碑状态”占用内存槽位，强制玩家进行资源规划与无害化回收。
